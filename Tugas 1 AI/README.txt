Jika menggunakan Windows :
1. Memakai bahasa pemrograman C++
2. Inisialisasi fungsi berada di SA.h
3. Implementasi fungsi berada di SA.cpp
4. Main program berada di main.cpp
5. Jika tidak bisa dibuild, maka coba untuk di rebuild terlebih dahulu

Jika menggunakan Linux :
1. Buka terminal
2. lalu ketik g++ main.cpp -o AISA dan tekan enter
3. kemudian ketik ./AISA dan tekan enter

NB : Saya membuat 2 versi karena awalnya saya membuat dilaptop saya yang OS nya windows, kemudian laptop saya rusak gamau nyala
     Jadinya saya minjem laptop temen saya yang kebetulan OS nya hanya linux.
